ВИ Добавить новости / мероприятия 

Admin   нажимает на кнопку добавить новости.
Admin нажимает сохранить :
проверяет на корректность заполнения текста
проверяет наличие текста:
выдает ошибку о том что нет текста
выдает ошибку о длине (длина не должна превышать 500 символов.)
 проверить на наличие нецензурных слов.(черный  список) 
добавляется  логин; Опубликовано + дата; Комментировать.
Текст сохраняется и выводится в блок новостей.

Admin нажимает на кнопку создать список « Темы для обсуждения »
Admin нажимает сохранить:
	1) проверяет наличие текста:
выдает ошибку о том что нет текста
	2) проверяет на корректность заполнения текста:
2.1 выдает ошибку о длине (длина не должна превышать 50 символов.)
2.2 проверить на наличие нецензурных слов.(черный  список) 
3) добавляется логин Опубликовано + дата; Комментировать.
4) каждый вопрос обсуждается в блоке комментировать.
5) данный список может заполняться как Admin так и пользователями данного блока, т.е он открыт для всех.

Admin нажимает на кнопку добавить  блок «Мероприятия»
Admin добавляет новости о предстоящих конференциях, встречах и т.д.:
проверяет наличие текста
выдает ошибку о том что текста нет
проверяет на корректность заполнения текста
выдает ошибку о длине заполнения текста(длинна не должна      превышать 500 символов)
проверить на наличие нецензурных слов(черный список)
Admin добавляет приглашение, где указано время, дата и место проведения мероприятия.
2.1 проверяет условия указанные выше
Так же выноситься отдельный блок о прошедших мероприятиях :
3.1 проверяются условия указанные выше
3.2 Опубликовано + дата; Комментировать. В коментах  обсуждается о результаты прошедших встреч.
