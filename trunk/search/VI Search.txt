ВИ Поиск проектов

1. User выбирает критерии поиска
2. Если поле имя пустое, тогда поиск по имени не ведется.
3. Поиск может осущевствлятся по тэгам.
4. Данные введены, User нажимает поиск.
 4.1 Поиск нашел данные
  4.1.1. Результат выводится списком по 20 на странице. 
  4.1.2. Скриншоты находятся под именем.
 4.2 Ничего не найдено
  4.2.1 Выводится что ничего не найдено.
  4.2.2 Предлагается корректировать запрос
