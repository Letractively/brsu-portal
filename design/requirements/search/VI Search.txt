ВИ Поиск проектов

1. User выбирает критерии поиска
2. Если поле имя пустое, тогда поиск по имени не ведется.
3. Поиск может осущевствлятся по тэгам.
4. Так же поиск:
  4.1 по дате создания
  4.2 по колву участников  
  4.3 по степени завершенности
  4.4 по используемой технологии(-ям)
5. Данные введены, User нажимает поиск.
 5.1 Поиск нашел данные
  5.1.1. Результат выводится списком по 20 на странице. 
  5.1.2. Скриншоты находятся под именем.
 5.2 Ничего не найдено
  5.2.1 Выводится что ничего не найдено.
  5.2.2 Предлагается корректировать запрос
