ВИ Сохранить историю проекта

1. ProjectManager нажимает изменить/сохранить проект
  1.1. Проект не существует
    1.1.1. создается отдельная запись для каждого поля 
  1.2. Проект существует
    1.2.1. ключевые поля проверяются на изменения
      1.2.1.1. изменения есть
        1.2.1.1.1. создается отдельная запись на каждое изменение 
      1.2.1.2. изменений нет
        1.2.1.2.1. ничего не происходит
>2. ключевые поля:
  2.1. Название проекта
  2.2. стадия разработки (прототип, альфа-, бетаверсия, ...)


ВИ Сохранить историю  участников проекта
Предусловие
1. ProjectManager нажимает добавить участника в проект
2. ProjectManager нажимает удалить участника из проекта
3. ProjectManager нажимает изменить полномочия участника проекта

  3.1 ключевые поля проверяются на изменения
    3.1 1. изменения есть
      3.1 1.1. создается отдельная запись для каждого измененого поля
    3.1.2. изменений нет
      3.1 2.1. ничего не происходит

1. ProjectManager нажимает изменить/сохранить профайл
  1.1. профойл не существует
